**Toast** — transient confirmation/alert. Hard border + offset shadow, colored icon chip.

```jsx
<Toast variant="success" title="Added to cart" message="Exhumed Rites Long Sleeve · M" onClose={dismiss} />
<Toast variant="error" title="Sold out" message="That size just went." />
```

Variants: `success` (lime) · `error` (blood) · `info` (pink).
