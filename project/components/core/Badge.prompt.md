**Badge** — small status stamp for products: NEW, LIMITED, SALE, SOLD OUT, PRE-ORDER. Mono uppercase with a hard black border.

```jsx
<Badge variant="new">New</Badge>
<Badge variant="limited">Limited · 100</Badge>
<Badge variant="soldout">Sold out</Badge>
<Badge variant="sale" dot>-30%</Badge>
```

Variants: `new`/`sale` (lime), `limited` (pink), `soldout` (black), `preorder` (paper), `blood` (red). `dot` adds a leading status dot.
